# pb-desarrollador-mobile
Prueba tecnica

# Instalación del proyecto (Front ionic)
npm i --force
# Ejecución del proyecto
npm start

# Generar APK
1: Instala la plataforma Android => ionic cordova platform add android
2: Construye tu app web => ionic build
3: Construye el APK => cordova build android --verbose
4: El APK generado queda en ==> platforms/android/app/build/outputs/apk/debug/app-debug.apk
